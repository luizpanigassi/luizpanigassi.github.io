Basics

  • Remember to ALWAYS link the CSS file to your document. Use <link rel="stylesheet" href="filename.css"> is the basic way to do it. But remember to put the right name, dummy.
  • Any fonts you use MUST BE LINKED. Remeber to use <link href="fontlink" rel="stylesheet">, then you can use it.
  • Borders required pixel size, solidity and a color.
  • Nice background image, right? You need to set the <body> to receive it in CSS. Remember to use the right link.
  • Oh shit! The background screwed my colors! Well... style them with CSS. Focus on <h>'s (from 1 to 6), and <p>'s.
  • Visual code is your new god. He will autocomplete your tags, help with linkage, and it's easy to find code errors. Enjoy.

Lists

For lists, you can use <ul> or <ol>, unordered or ordered lists. Open the tag, then insert the <li> in it, which are the List Items. You can style the lists in CSS, go crazy. If you type "style" between brackets you'll see a dozen things you can do to your list. I used armenian in mine, makes no sense whatsoever.

Containers

So, you can created containers. Here's where <div>s can help. You can set a div, give it a class, then style it in CSS using borders. A nice border, with some padding, can make things separate like boxes, like those pretty babies here. Remember you can use border-top, border-bottom, left and right.

Colors

To add colors, you have to use the color attribute. Then, remember, you can use the name, color code, or rgb. Here are some examples of colors you can use:

Remember, you can always use rgbA to input an alpha value, which refers to opacity.

Fonts

Fonts are easy. Just go to Google Fonts, choose your font, its weight, copy the code, find the <head> section and paste it, so the fonts can be loaded. Then just enter the font-family code in CSS, and BOOM!

Other codes to pay attention to in CSS are font-weight (for boldness), transform (for uppercase and such), size (for what again?) and style (for italics)

Tables

Tables can be annoying. First, set the <table>, to wrap everything. Inside it, you need to set the <thead>, to set the header. Inside it, you can input the <tr>, which are the row. Inside the row, input the <td>, and each symbolizes a cell in the row, generating new columns.

When you're done with the head, go for <body>, following the same idea of the head. Why separate, and not just input TR and TD? Because then you can style everything individually using CSS!
